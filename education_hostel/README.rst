Educational Hostel Management
=============================
To efficiently manage the entire residential facility in the school, you need
to implement effective systems and procedures. This involves coordinating
various aspects such as room assignments, maintenance, security, and student
welfare.

Configuration
=============
- Need to configure the Education Hostel user group from the user Configuration
